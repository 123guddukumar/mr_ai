Master API Documentation

Complete technical reference for Developers to build white-label AI platforms.

Admin API App/Client API User API Chat & Usage
Hierarchy Overview: Admin (Creates Apps) → App/Client (Creates Users & Agents) → End User (Manages their own Agents).
1. Super Admin Level (X-Admin-Token)

Admin uses these APIs to create and manage full White-Label Apps (Clients).

Create New App (Client)
POST /api/admin/clients/create-direct
FieldTypeRequiredDescription
namestringYesOwner's Name
emailstringYesLogin Email
passwordstringYesLogin Password (min 6 chars)
business_namestringNoThe App's Brand Name
categorystringNoe.g. "SaaS", "Education"
gst_numberstringNoTax ID
website_urlstringNoApp's website
logo_urlstringNoURL to brand logo
{
  "name": "Guddu Kumar",
  "email": "app@example.com",
  "password": "securepassword123",
  "business_name": "Vectorize AI App",
  "category": "Technology",
  "gst_number": "27AAAAA0000A1Z5",
  "city": "Mumbai",
  "address": "Tech Park, HSR Layout"
}
2. App / Client Level (X-App-Token)

The App uses these APIs to manage its own ecosystem of users, agents, and datastores.

Create Sub-User
POST /api/clients/sub-users
FieldTypeRequiredDescription
namestringYesUser's Full Name
emailstringYesUser's Login Email
passwordstringYesUser's Password
professionstringNoUser's Job Role
user_typestringNo"New", "Prime", "Demo"
Create DataStore
POST /api/datastores
FieldTypeRequiredDescription
namestringYesKnowledge Base Name
Create AI Agent
POST /api/agents
FieldTypeRequiredDescription
namestringYesBot Name
datastoresarrayNoList of DataStore IDs to link
personalitystringNoHow the bot should behave
starting_messagestringNoFirst message to user
voice_configobjectNoVoice provider settings
{
  "name": "Customer Support Bot",
  "datastores": ["ds_12345", "ds_67890"],
  "personality": "Helpful, professional and empathetic",
  "starting_message": "Hello! I am your AI assistant. How can I help you?",
  "voice_config": {
    "provider": "elevenlabs",
    "voice_id": "pNInz6obpg8n9SaBeCm1"
  }
}
3. End User Level (User X-App-Token)

End Users manage their own agents and datastores using the same endpoints as the Client, but within their own scope.

Upload Data (PDF/Web/YT)
POST /api/datastores/{ds_id}/upload-pdf
Requirement: Form-Data with file field.
Scrape Website
POST /api/datastores/{ds_id}/ingest-url
{ "url": "https://example.com/docs" }
4. Chat & Interaction
Ask Agent (Chat)
POST /api/agents/{agent_id}/ask
{
  "question": "What is the return policy?",
  "history": [
    {"role": "user", "content": "Hi"},
    {"role": "assistant", "content": "Hello!"}
  ]
}
QR Code Generation
GET /api/agents/{agent_id}/qr

Returns a QR image that opens the agent chat interface.
