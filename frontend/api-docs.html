🔐 Authentication

All data endpoints require an X-API-Key header. Public endpoints (health, key status, key validate) do not require authentication.

X-API-Key: mrairag-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
⚠️ Store your API key securely. It is shown only once when generated and cannot be retrieved again.

Protected Routes Requires X-API-Key

📄 PDF Upload

POST /api/upload Auth Upload & index a single PDF

Upload a PDF file (max 50MB). The file is parsed, chunked, embedded and stored in the vector database for querying.

Parameters

NameTypeDescription
fileForm/FilePDF file (multipart/form-data)
X-API-KeyHeaderYour API key

Try It

Code Examples

curlPython
curl -X POST http://localhost:8000/api/upload \
  -H "X-API-Key: mrairag-YOUR_KEY" \
  -F "file=@document.pdf"

🤖 Ask a Question

POST /api/ask Auth Query indexed documents with LLM

Ask a natural language question. The system retrieves relevant chunks from indexed sources and generates an answer using the active LLM provider.

Request Body (JSON)

FieldTypeDescription
questionstringYour question (3–2000 chars)
top_kint (opt)Number of chunks to retrieve (1–20, default 5)

Try It

Code Examples

curl -X POST http://localhost:8000/api/ask \
  -H "X-API-Key: mrairag-YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{"question": "What are the key findings?", "top_k": 5}'

📺 YouTube Ingestion

POST /api/ingest-youtube Auth Ingest YouTube video transcript

Provide any YouTube URL. The system fetches transcripts using a 3-stage fallback: youtube-transcript-api → yt-dlp subtitles → Whisper audio transcription.

Request Body

FieldTypeDescription
urlstringFull YouTube URL
whisper_modelstring (opt)tiny|base|small|medium|large (default: base)

Try It

curl -X POST http://localhost:8000/api/ingest-youtube \
  -H "X-API-Key: mrairag-YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{"url": "https://youtu.be/dQw4w9WgXcQ"}'

🎬 Video File Upload

POST /api/ingest-video Auth Upload video/audio → Whisper transcription

Upload any video or audio file (mp4, mkv, webm, mov, avi, mp3, wav, m4a, etc.) up to 500MB. Whisper AI transcribes the audio into indexed text.

Parameters

NameTypeDescription

🐍Python Integration

Full workflow: upload PDF → ask question

import requests

BASE_URL = "http://localhost:8000/api"
API_KEY  = "mrairag-your-key-here"
HEADERS  = {"X-API-Key": API_KEY}

# 1. Upload a PDF
with open("document.pdf", "rb") as f:
    r = requests.post(f"{BASE_URL}/upload",
                      headers=HEADERS,
                      files={"file": f})
    print(r.json())  # {"success": true, "total_chunks": 42, ...}

# 2. Ask a question
r = requests.post(f"{BASE_URL}/ask",
    headers={**HEADERS, "Content-Type": "application/json"},
    json={"question": "What are the key findings?", "top_k": 5})
print(r.json()["answer"])

# 3. Ingest a YouTube video
r = requests.post(f"{BASE_URL}/ingest-youtube",
    headers={**HEADERS, "Content-Type": "application/json"},
    json={"url": "https://youtu.be/dQw4w9WgXcQ"})
print(r.json())

# 4. Scrape a website
r = requests.post(f"{BASE_URL}/ingest-url",
    headers={**HEADERS, "Content-Type": "application/json"},
    json={"url": "https://en.wikipedia.org/wiki/AI"})
print(r.json())
🌐JavaScript / Node.js Integration

Works in browser (fetch) or Node.js

const BASE_URL = 'http://localhost:8000/api';
const API_KEY  = 'mrairag-your-key-here';
const headers  = { 'X-API-Key': API_KEY };

// 1. Ask a question
async function askQuestion(question) {
  const res = await fetch(`${BASE_URL}/ask`, {
    method: 'POST',
    headers: { ...headers, 'Content-Type': 'application/json' },
    body: JSON.stringify({ question, top_k: 5 })
  });
  const data = await res.json();
  return data.answer;
}

// 2. Upload a PDF (browser)
async function uploadPDF(file) {
  const form = new FormData();
  form.append('file', file);
  const res = await fetch(`${BASE_URL}/upload`, {
    method: 'POST',
    headers,     // Don't set Content-Type — let browser set boundary
    body: form
  });
  return await res.json();
}

// Usage
askQuestion('Summarize the document').then(console.log);
